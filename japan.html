Day

02
Day
关西人文 · 大阪 → 奈良 → 京都
Osaka → Nara → Kyoto
游览大阪城公园，春日樱花与历史古城相映成趣；前往奈良公园，零距离喂食自由漫步的萌鹿，参拜千年古社春日大社；移步古都京都，拾级而上至名列世界文化遗产的清水寺，在山间舞台俯瞰京都城的春日画卷。晚间体验抹茶制作，感受日式茶道精髓。
✓ 大阪城公园 ✓ 奈良公园（喂鹿） ✓ 春日大社 ✓ 清水寺 ✓ 抹茶体验 🍵 京都特色料理
03
Day
京都 → 心斋桥 · 温泉旅馆
Kyoto → Osaka · 温泉升级
漫步心斋桥与道顿堀，体验大阪最热闹的美食与购物街区，品尝章鱼烧与大阪烧；下午前往特色温泉旅馆，在天然温泉中放松身心，享用精心准备的温泉晚宴，度过最纯粹的日式旅馆之夜。
✓ 心斋桥 ✓ 道顿堀美食街 ♨️ 温泉旅馆入住（升级） 🍜 温泉晚宴 🛍 综合免税店 约60分钟
04
Day
富士山 · 忍野八海 · 伊豆河津樱
Mt. Fuji → Oshino → Kawazu
抵达富士山五合目（海拔2,305米），在云端俯瞰四周火山地貌与关东平原；游览名水百选忍野八海，在富士山雪融水汇聚的清澈池塘旁感受纯净自然；下午奔赴此行最重要的限定目的地——伊豆河津町，沿着800株深粉河津樱构成的浪漫樱花大道漫步，赏全日本最早盛开的早樱。
✓ 富士山五合目 ✓ 忍野八海 🌸 伊豆河津樱花大道（季节限定） 🍱 富士特色料理
05
Day
东京风尚 · 银座 · 台场
Tokyo · 都市风情
清晨漫步大横川散步道，欣赏早樱盛放的江户水岸风光（季节限定）；参拜樱神宫，祈福打卡这座以樱花命名的古老神社；下午穿梭于时尚与古典并存的银座，感受东京最奢华的购物街区；傍晚前往台场，这座东京湾上的人工岛囊括了彩虹大桥夜景、娱乐设施与海景餐厅，是此行最适合拍照留念的压轴地点。
🌸 大横川散步道（早樱·季节限定） ✓ 樱神宫 ✓ 银座 ✓ 台场 🛍 长岛奥特莱斯 约60分钟 🍱 东京特色餐

重要注意事项 · Important Notices

🖐 入境指纹采集

  • 入境日本必须接受双食指指纹采集及面部拍照
  • 拒绝配合指纹采集者将不获准入境，且不退任何费用
  • 请提前告知同团旅客，避免产生不必要的麻烦

🏨 日本酒店须知

  • 日本酒店房间紧凑，寸土寸金
  • 请勿与国内同级酒店面积作对比
  • 温泉旅馆着浴衣使用公共温泉，请留意礼仪规范

🚌 巴士服务规定

  • 日本法律规定司机每天行车不超过10小时
  • 因游客迟到导致需延长巴士服务，需支付 ¥10,000日元/小时 延长金
  • 请务必准时集合，避免影响全团行程安排

⚠️ 脱团与保证金

  • 擅自脱团属于违法行为，违规金 ¥1,000元/天/人
  • 出团前须交纳归国保证金 ¥50,000元/人，按时随团归国后全额退还
  • 本行程作为《团队出境旅游合同》不可分割的一部分
